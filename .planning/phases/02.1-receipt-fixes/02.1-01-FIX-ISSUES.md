### UAT-001: Price input auto-formats on keystroke, preventing decimal entry

**Discovered:** 2026-01-14
**Resolved:** 2026-01-14
**Phase/Plan:** 02.1-01-FIX
**Severity:** Blocker
**Feature:** Price input editing
**Description:** When typing a price, the field auto-formats to "X.00" immediately after the first digit, moving the cursor to the end. This prevents proper decimal entry.
**Root Cause:** `value={item.price.toFixed(2)}` reformatted on every keystroke, causing cursor to jump.
**Resolution:** Changed to format on blur only. Added local state for input fields that stores raw string during editing, then parses and formats on blur.
**Commit:** d43f20f

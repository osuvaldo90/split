<task type="auto">
  <name>Task 1: Create Vite + React + TypeScript project with Convex</name>
  <files>package.json, vite.config.ts, tsconfig.json, src/main.tsx, src/App.tsx, convex/</files>
  <action>
Create the project using Convex's recommended approach:

1. Run `npm create convex@latest split` in the parent directory (will create split/ subdirectory)
   - OR if already in split directory: `npm create vite@latest . -- --template react-ts` then `npm install convex`

2. If using manual approach:
   - Initialize vite with react-ts template
   - Install convex: `npm install convex`
   - Run `npx convex dev` to initialize Convex project (creates convex/ directory with _generated/)

3. Update src/main.tsx to wrap App with ConvexProvider:
```tsx
import { StrictMode } from 'react';
import { createRoot } from 'react-dom/client';
import { ConvexProvider, ConvexReactClient } from 'convex/react';
import App from './App';
import './index.css';

const convex = new ConvexReactClient(import.meta.env.VITE_CONVEX_URL);

createRoot(document.getElementById('root')!).render(
  <StrictMode>
    <ConvexProvider client={convex}>
      <App />
    </ConvexProvider>
  </StrictMode>
);
```

4. Create a minimal App.tsx that displays "Split" to verify the setup works.

AVOID: Don't use create-react-app (deprecated). Don't skip the ConvexProvider setup - we need real-time from day one.

  </verify>

```tsx

import { api } from "../convex/_generated/api";

function App() {

  return (
    <div className="min-h-screen bg-gray-100 flex flex-col items-center justify-center gap-4">
      <h1 className="text-3xl font-bold text-gray-900">Split</h1>

    </div>
  );
}
export default App;
```
